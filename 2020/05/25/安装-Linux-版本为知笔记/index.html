安装 Linux 版本为知笔记

官网安装方式

  • 下载 http://www.wiz.cn/wiznote-linux.html
  • Ubuntu 18.04 下运行前，请先执行 sudo apt install openssl1.0 libssl1.0-dev ，再双击运行绿色版
  • 推荐使用 ibus 输入法

下载 .deb 文件进行安装

  • 安装包下载 https://launchpad.net/~wiznote-team/+archive/ubuntu/ppa/+packages
  • 双击下载的 .deb 文件进行安装
